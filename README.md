## ✨ ما الذي يميز هذه النسخة؟

هذه نسخة مخصصة (Fork) من مشروع Harbor الأصلي، طُوِّرت بالإضافات التالية:

| الميزة | الوصف |
|---|---|
| 🇸🇦 **العربية أولًا** | اللغة العربية هي اللغة الافتراضية مع دعم كامل للكتابة من اليمين لليسار (RTL) |
| 💚 **ثيم ABADIOSA** | ثيم أخضر زمردي خاص هو المظهر الافتراضي للتطبيق، مع بقاء كل الثيمات الأصلية متاحة |
| 🪟 **هوية خاصة** | عنوان النافذة والتطبيق يحمل اسم النسخة الخاصة |
| 🔄 **تحديثات مستقلة** | نقطة التحديث التلقائي تشير إلى هذا المستودع بدل المشروع الأصلي، فلا تُستبدل تخصيصاتك أبدًا |

## 🚀 أبرز مزايا Harbor الأصلية

- **مشغّل أصلي (Native)** مبني على mpv مع دعم DVR وصورة داخل صورة (PiP)
- **محرك ترتيب مصادر البث** لفرز وترتيب الروابط تلقائيًا
- **حفلات مشاهدة جماعية** (Watch Parties)
- **قنوات بث مباشر** وأنمي وبيانات من Wikidata
- **محرك ثيمات كامل** مع استوديو لتصميم ثيمك الخاص
- يعمل مع **حسابك في Stremio** وإضافاتك الحالية مباشرة

## 🛠️ البناء من المصدر

المتطلبات: Node.js 22+‎ و pnpm و Rust (لبناء تطبيق سطح المكتب عبر Tauri).

```bash
# تثبيت الاعتماديات
pnpm install

# جلب الملفات الثنائية (mpv, ffmpeg, yt-dlp, الخطوط)
pnpm run setup

# تشغيل نسخة التطوير
pnpm run tauri dev

# بناء نسخة نهائية
pnpm run tauri build
```

كما يمكن بناء حِزم التثبيت لويندوز وماك ولينكس عبر GitHub Actions: من تبويب **Actions** في المستودع شغِّل ورك فلو **Tauri Build** يدويًا.
